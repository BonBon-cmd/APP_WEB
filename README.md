# Lab01 - Nhóm 27

Dự án gồm 3 phần: Static Web Server, HTTP Client, Network Monitoring.

## Cấu trúc thư mục
- server.js: Express server
- client.js: HTTP client
- monitor.js: Script giám sát mạng
- public/: Static files (HTML, CSS, JS)
- screenshots/: Ảnh chụp màn hình
- docs/: Báo cáo phân tích
- presentation/: Slide trình bày

## Hướng dẫn cài đặt
1. Cài đặt các gói phụ thuộc:
    ```bash
    npm install
    ```
2. Chạy ứng dụng:
    ```bash
    npm start
    ```
    ```bash
    node client.js
    ```
    ```bash
    node monitor.js
    ```
